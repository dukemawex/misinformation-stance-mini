# Misinformation Stance Mini

Compact stance-detection model for misinformation claims.

## Getting started

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m misinformation_stance_mini
```

## Project structure

```
misinformation_stance_mini/        # source package
tests/      # tests
requirements.txt
```
